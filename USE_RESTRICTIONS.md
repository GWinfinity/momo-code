# momo Code Use Restrictions

momo Code is open-source software licensed under the MIT License. In addition to the MIT License terms, the following use restrictions apply:

## Prohibited Uses

You may not use momo Code, or any models fine-tuned through momo Code's `/fine-tune` feature, for:

1. **Illegal Activities** -- Any purpose that violates applicable laws or regulations
2. **Fraud or Deception** -- Generating deceptive content, impersonation, or social engineering
3. **Harm to Individuals** -- Harassment, discrimination, or creation of non-consensual intimate imagery
4. **Military Applications** -- Development of weapons systems, targeting systems, or military surveillance
5. **Malicious Cyber Activities** -- Creating malware, ransomware, phishing tools, or credential-stealing tools
6. **Unauthorized Data Collection** -- Scraping private data without consent, bypassing access controls
7. **High-Risk Autonomous Operations** -- Fully automated operations in safety-critical domains (medical, financial, infrastructure) without human oversight

## Self-Evolution Restrictions

When using the `/fine-tune` self-evolution feature:
- Models may only be trained on data you have rights to use
- Fine-tuned models must not be used for any of the prohibited purposes above
- Human review is required before promoting a fine-tuned model to production
- You must maintain audit logs of training data and promotion decisions

## Enforcement

Violations of these restrictions may result in:
- Termination of access to momo Code services
- Legal action under applicable law
- Reporting to relevant authorities
